# Smart Utility

Console ilova yarating. Dastur ishga tushganda foydalanuvchiga asosiy menyu chiqsin va u tanlov asosida turli bo‘limlarga kirsin. Har bir bo‘limdan chiqishda foydalanuvchi asosiy menyuga qaytish yoki dasturni yopish imkoniga ega bo‘lsin.

## Asosiy kommandalar

1. **Number Analyzer**
2. **Geometry Zone**
3. **Text Tools**
4. **Exit**

### 1.0 Number Analyzer 

Number Analyzer bo'limida sonlarning juft toqligi,musbat manfiyligi va kvadrati, kubini topishga yordam beradi.
1. Son juft yoki toqligini aniqlash
2. Son musbat, manfiy yoki nol 
3. Sonning kvadrati va kubi
4. Exit

#### 1.1 Son juft yoki toqligini aniqlash

Console dan biror son kirib keladi va shu sonning juft yoki toqligini topib natijani qaytaradi.

#### 1.2 Son musbat, manfiy yoki nol

Consoledan biror son kirib keladi. Kiritilgan son noldan katta bo'lsa musbat, kichik bo'lsa manfiy va nolga teng bo'lsa nol deb chiqarib beradi.

#### 1.3 Sonning kvadrati va kubi

Consoledan biror son kiritiladi va qaysi darajaga oshirilishi yoziladi. Kvadrati belgilansa kiritilgan sonning kvadratini yoki kubi belgilansa sonning kubini hisoblab natijani chiqarib beradi.

### 2.0 Geometry Zone

Foydalanuvchi Consoledan shakl tanlaydi, dastur ASCII rasm va uning yuza formulasini chiqaradi.

1. Rectangle
2. Triangle
3. Circle
4. Exit

#### 2.1 Reactangle

Console dan Agar foydalanuvchi Rectangle ni tanlasa, to‘rtburchak shaklini chiqaradi. Shundan so‘ng uning yuza formulasi — Area = a × b ni ko‘rsatadi.

#### 2.2 Triangle


Console dan Foydalanuvchi Triangle ni tanlasa uchburchak shaklini chiqaradi va uning yuza formulasini ko‘rsatadi.

#### 2.3 Circle

Agar foydalanuvchi Console dan Circle ni tanlasa doira shaklini chiqaradi va uning yuza formulasini ko‘rsatadi.

### 3.0 Text Tools

Console dan foydalanuvchi biror so‘z yoki matn kiritadi. So‘ng menyudan kerakli amalni tanlaydi.

1. **Matn uzunligini aniqlash**
2. **Matnni katta harflarga o‘tkazish**
3. **Matnni teskari chiqarish**
4. **Exit**

#### 3.1 Matn uzunligini aniqlash 

Foydalanuvchi kiritgan matndagi barcha belgilar sonini hisoblab beradi.

#### 3.2 Matnni katta harflarga o'tkazish

Foydalanuvchi kiritgan matndagi barcha harflarni katta harf ko‘rinishiga o'zgartirib chiqaradi.

#### 3.3 Matnni teskari chiqarish

Foydalanuvchi kiritgan matnni oxiridan boshlab teskari tartibda chiqaradi.

### 4 Exit

Console dan Exitni tanlasa dastur yakunlanadi va hayrlashadi.
